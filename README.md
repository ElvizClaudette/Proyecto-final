# Proyecto-final

## Etapa 1.Descripción del problema
Crear una aplicacion en Java en donde se muestren ejemplos de las estructuras de control vistas en la materia 
Las estructuras son:
- if else
- switch
- for
- while

## Etapa 2.Especificaciones
1. Entrada:
  - For:
    - Ejemplo 1:
       - int numero
    - Ejemplo 2:
       - int mes
    - Ejemplo 3:
       - int xumero
       - int numero
  - While
    - Ejemplo 1:
       - int i
    - Ejemplo 2:
       - int contador
       - int fin
    - Ejemplo 3: 
       - int decimal
       - String binario
  - If else:
    - Ejemplo 1:
       - int tem
    - Ejemplo 2: 
       - int dis
    - Ejemplo 3:
       - int num
       - int div
  - Switch:    
       - Menu
2. Proceso:

  - Mostrar ejemplos de estructuras de control

3. Salida:
~~~
 +--------------------------------------------------+
 |         MENÚ DE EJEMPLOS DE PROGRAMACIÓN         |
 +--------------------------------------------------+
 | 1. EJEMPLO DE IF ELSE:                           |
 | 2. EJEMPLO DE SWITCH:                            |
 | 3. EJEMPLO DE FOR:                               |
 | 4. EJEMPLO DE WHILE:                             |
 | 5. SALIR                                         |
 +--------------------------------------------------+
~~~
